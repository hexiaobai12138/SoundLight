# SoundLight

#### 介绍

华中科技大学计算机学院 2021 级软件工程项目课程设计
闯关剧情类游戏，激发对视障人士的关怀。

#### 游戏玩法

一共四关，从第一关跟着剧情完成关卡即可进入下一关。
WASD进行角色移动
Space（空格）敲盲杖（发出声波）
R键进行交互（关卡题记出现后按R键进入关卡）
注意角色有生命值与受伤系统，请不要轻易撞击npc与障碍物。
